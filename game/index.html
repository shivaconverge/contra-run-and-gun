RUN & GUN — VERTICAL SLICE

←/→ or A/D move  ·  ↑ aim  ·  ↓ prone (duck fire)  ·  Z / Space jump  ·  X fire  ·  R restart  ·  1 Arcade / 2 Casual  ·  M mute
Arcade (default): one hit = one life — the 1987 way. Casual: a shield + extra lives. Grab weapon capsules — Spread, Machine, Laser (pierces!), Fire (corkscrew) (lost on death). Defeat the SENTINEL boss — hold ↓ to duck its cannon fire.
Real pixel-art sprites + tileset; procedural fallback for any asset not yet in assets/.

ROTATE TO LANDSCAPE Run & Gun plays in widescreen — turn your device sideways.

▶ SERVE ME TO PLAY

This build uses ES modules, which browsers only load over http(s) — not from a local file:// path. One command, no install (just Node): 
cd game && npm start
then open http://localhost:8080/.
No Node? python3 -m http.server 8080 in the game/ folder.

If you opened a hosted link and still see this, a script failed to load — check the browser console.
