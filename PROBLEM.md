# Problem

Build and continuously ship a production-ready, browser-playable run-and-gun action game in the Contra lineage that real players prefer over today's popular 2D Contra-like games (web/Android, last 3-5 years) and that matches or exceeds their visual and gameplay fidelity while capturing the nostalgic feel of the original arcade Contra.

This git repo is the solution. `master` is the integrated deliverable; each loop works on its own branch and opens PRs upward.
